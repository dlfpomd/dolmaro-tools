🟢 자동 실행 (기본)

매일 오전 10시와 오후 3시에 PC가 자동으로 네이버 블로그(dlfpomd2)에서 미이관 글 1개를 가져와 이레한의원 홈페이지에 발행합니다. 발행되면 이 페이지 목록에 자동 반영 (새로고침).

작업 스케줄러 이름: IREA_Blog_AM, IREA_Blog_PM

▶️ 지금 바로 실행하고 싶을 때

방법 A — 탐색기에서 더블클릭 (제일 쉬움):

  1. 파일 탐색기를 엽니다 (Win + E)
  2. 주소창에 이 경로를 붙여넣고 엔터:
    C:\Users\user\Documents\Claude\blog_migrator
  3. auto_migrate.bat 을 더블클릭
  4. 1개 글이 이관된 뒤 auto_migrate_log.txt에 완료 시각이 기록됨

방법 B — 작업 스케줄러에서 즉시 실행:

  1. Win + R → taskschd.msc 엔터
  2. IREA_Blog_AM 또는 IREA_Blog_PM 우클릭 → "실행"

방법 C — PowerShell에서 여러 개 한 번에:

cd C:\Users\user\Documents\Claude\blog_migrator
python main.py --batch 5

💡 --batch N의 N은 한 번에 이관할 글 개수. 너무 크게 하면 Claude API 비용이 많이 듭니다.

✏️ 설정(블로그·카테고리·모델 등) 수정

아래 파일을 메모장 또는 VSCode로 열어 편집:

  • C:\Users\user\Documents\Claude\blog_migrator\config.py

주요 항목:

  • NAVER_BLOG_ID — 가져올 네이버 블로그 아이디 (현재 dlfpomd2)
  • START_YEAR — 이 연도 이후 글만 수집 (현재 2020)
  • SORT_ORDER — oldest(오래된 것 먼저) / newest(최신 먼저)
  • WP_CATEGORY_ID — 홈페이지 카테고리 (현재 3 = 이레칼럼)
  • WP_POST_STATUS — publish(즉시 공개) / draft(임시저장)
  • CLAUDE_MODEL — 제목·Q&A 생성에 쓰는 Claude 모델

⚠️ 보안 주의: config.py에 ANTHROPIC_API_KEY와 WP_APP_PASSWORD가 평문으로 들어있습니다. 절대 GitHub 등 공개 장소에 올리지 마세요. PC 분실/공유 시 키 회전 필수.

🆕 새 PC에 세팅할 때 (최초 1회)

이미 세팅된 PC에서는 건너뛰세요.

  1. Python 3.10+ 설치 (python.org) — "Add Python to PATH" 체크
  2. 폴더 복사 — 기존 PC의 C:\Users\user\Documents\Claude\blog_migrator\ 전체를 새 PC 같은 경로에 복사
  3. 의존성 설치 — PowerShell에서: 
    cd C:\Users\user\Documents\Claude\blog_migrator
pip install -r requirements.txt
  4. 스케줄러 등록 — setup_scheduler.bat을 우클릭 → "관리자 권한으로 실행"
  5. 테스트 — auto_migrate.bat 더블클릭 → auto_migrate_log.txt에 "Done" 기록 확인

🔎 문제가 생겼을 때

  • 실행 로그: C:\Users\user\Documents\Claude\blog_migrator\auto_migrate_log.txt
  • 이관 이력: C:\Users\user\Documents\Claude\blog_migrator\processed_posts.json (이 파일에 등록된 글은 다음 실행에서 건너뜀)
  • 스케줄 확인: schtasks /query /tn "IREA_Blog_AM"
  • 스케줄 해제: schtasks /delete /tn "IREA_Blog_AM" /f 와 /tn "IREA_Blog_PM"
  • 같은 글 다시 이관하고 싶을 때: processed_posts.json에서 해당 log_no 항목 삭제 후 재실행
  • Claude API 크레딧 부족: console.anthropic.com/settings/billing
  • 워드프레스 인증 실패: irea.co.kr 관리자 → 사용자 → 새 앱 비밀번호 발급 → config.py의 WP_APP_PASSWORD 교체
