Outlier Score란?

해당 영상의 조회수를 채널의 최근 영상 평균 조회수로 나눈 값입니다.
예시: 채널 평균 조회수 5만인데 특정 영상이 500만 조회 → Outlier Score = 100x

1~3x: 보통
3~10x: 주목할만함
10~50x: 트렌드
50x+: 바이럴

API 설정

YouTube Data API v3

YouTube 트렌드 데이터를 가져오기 위해 API 키가 필요합니다.

API 키 발급 방법

  1. Google Cloud Console에 접속
  2. 새 프로젝트 생성 또는 기존 프로젝트 선택
  3. "API 및 서비스" → "라이브러리"에서 YouTube Data API v3 활성화
  4. "사용자 인증 정보" → "사용자 인증 정보 만들기" → "API 키"
  5. 생성된 키를 위 입력란에 붙여넣기
