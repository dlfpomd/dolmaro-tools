토킹 영상의 빈 구간을 자동으로.

mp4/mov 영상을 넣으면 조용한 구간(무음)을 잘라(점프컷) 곧바로 CapCut 드래프트로 만들어 줍니다. 끄고 켜면 CapCut 목록에 바로 떠요. 명령어 입력은 필요 없습니다. 영상은 PC를 벗어나지 않습니다.

⬇ 다운로드 (Windows) .zip 압축 풀고 start.bat 더블클릭

무엇을 하나

· 무음 컷

조용한 구간을 감지해 점프컷. 길이/감도는 슬라이더로 조절.

· CapCut 자동 등록

드래프트를 만들고 목록에 바로 등록 — 껐다 켜면 보입니다.

· 100% 로컬

영상이 외부로 전송되지 않고 PC 안에서만 처리됩니다.

· 명령어 불필요

start.bat 더블클릭 → 브라우저에서 드래그&드롭.

3단계로 시작

  1. 다운로드 · 압축 풀기. 위 버튼으로 받아 마우스 오른쪽 → 압축 풀기.
  2. start.bat 더블클릭. 처음 한 번 파이썬이 필요합니다(안내가 자동으로 떠요). 필요한 구성요소는 자동 설치됩니다.
  3. 브라우저에서 편집. 창이 자동으로 열리면 영상을 끌어다 놓고 옵션 체크 후 “처리 시작”. 끝나면 CapCut 목록에 드래프트가 생깁니다.

필요한 것

Windows · CapCut(캡컷) 설치(한 번 실행해 둘 것) · Python(없으면 설치 안내가 자동으로 열립니다). 인터넷은 최초 1회 구성요소 설치 때만 필요합니다.

※ 드래프트 처리 전에는 CapCut을 꺼두세요. 처리 후 CapCut을 켜면 목록 맨 위에 뜹니다. 검증은 항상 CapCut에서 직접 재생으로.

왜 웹에서 바로 안 돌아가나요?

이 도구는 님 PC에서 영상을 직접 처리하고, 결과를 CapCut 드래프트 폴더에 파일로 저장합니다. 브라우저는 보안상 PC 폴더에 파일을 쓸 수 없고 무거운 AI 모델을 돌리기 어렵기 때문에, 다운로드해서 실행하는 방식으로 만들었습니다. (모든 처리는 님 PC 안에서만 일어나며, 영상이 외부로 업로드되지 않습니다.)

돌마로 툴즈 · 캡컷 에이전트 — 로컬 실행, 영상은 PC를 벗어나지 않습니다.
